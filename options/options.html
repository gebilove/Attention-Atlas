网站提示词管理

一个提示词可以关联多个网站。网站请填写完整 origin，例如 https://example.com。

说明

内容脚本只发送知识块文本和当前模式，不会把 API key 注入网页。请求由扩展 background service worker 发出。

全局默认提示词会应用到所有网页；网站提示词按 origin 关联，例如同一个提示词可以同时作用于多个课程站点，并且优先级高于全局默认提示词。

OpenAI 会调用 Chat Completions，使用 Authorization: Bearer。Anthropic 会调用 Messages API，使用 x-api-key 与 anthropic-version。

切换 Provider 会自动填入对应默认 Endpoint 和 Model；如果你填了自定义代理地址，切换时不会覆盖。
